A função rankx é utilizada para rankear os valores de acordo com uma coluna passada como primeiro parâmetro.
Importante: Coluna e não tabela!

Nesse caso, o rank esta sendo realizado com base na coluna com o nome das lojas.

Fórmula para identificar a quantidade vendida: Quantidade vendida = SUM(fVendas[Quantidade])
Fórmula para rankear a quantidade vendida de acordo com cada loja: Rank quantidade vendida = RANKX(all(fVendas[Loja]),[Quantidade vendida])
Fórmula para rankear somente os 10 primeiros ranks: Rankx ate 10 = IF(HASONEVALUE(fVendas[Loja]),IF([Rank quantidade vendida]<=10,[Rank quantidade vendida],BLANK()))
